## 🛠️ How to Compile and Run

Use **GCC compiler**:

```bash
gcc program.c -o program
./program
```

---

## 🐞 Debugging with GDB

You can debug your C programs using **GDB (GNU Debugger)**.

### 🔗 GDB Tutorial Link: 

👉 Online GDB File Folder : https://onlinegdb.com/QmTeGqWIN


### Basic Commands:

```bash
gcc -g program.c -o program
gdb ./program
```

Inside GDB:

```
run
break main
next
print variable_name
```
